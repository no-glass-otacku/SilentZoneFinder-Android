## 1️⃣ Background and Motivation (배경 및 동기)

### 📱 앱 소개

**SilentZoneFinder**는 조용한 장소를 찾아주는 안드로이드 앱입니다. 
사용자가 직접 소음 수준을 측정하여 리뷰를 남기고, 지도에서 주변의 조용한 카페, 식당, 도서관 등을 쉽게 찾을 수 있도록 도와줍니다.

### 핵심 가치
- **실제 소음 측정**: 사용자가 직접 측정한 정확한 소음 데이터
- **시각적 지도**: 소음 수준별 색상 마커로 한눈에 파악
- **커뮤니티 기반**: 모든 사용자가 함께 만드는 조용한 장소 데이터베이스

### 💡 개발 동기
- 조용한 작업 공간을 찾는 수요 증가 (재택근무, 공부, 집중 작업)
- 기존 리뷰 앱의 소음 정보 부족 및 주관적 평가의 한계
- 실제 측정 데이터 기반의 객관적 소음 정보 제공 필요

### 📝 제안서 대비 변경사항
- **추가된 기능**:
  - ✅ 날씨 정보 연동 (OpenWeatherMap API)
  - ✅ Noise Trend Today 그래프 (시간대별 소음 추이)
  - ✅ 검색 기록 및 자동완성 기능
  - ✅ 알림 시스템 (WorkManager 기반 백그라운드 작업)
  - ✅ 소음 임계값 알림 기능
- **변경된 기술**:
  - Firebase → Supabase로 변경 (인증, 데이터베이스, 스토리지)
  - Room Database 추가 (로컬 캐싱 준비)
  - Fragment 활용 (DialogFragment, BottomSheetDialogFragment)

## 3️⃣ Application Features (애플리케이션 기능) ⭐ 가장 중요!

### ✅ 기능 작동 여부

- ✅ 지도 마커 표시 및 필터링 (즐겨찾기 전용 마커 스타일은 진행 중)

- ✅ 이미지 업로드 및 표시
- ✅ 날씨 정보 표시

### 🎯 주요 기능

### 1. 지도 화면 (MainActivity)

#### 1.1 실시간 소음 수준 표시
- **기능 설명**: 지도에 표시된 장소들의 평균 소음 수준을 색상으로 구분하여 표시
- **소음 수준 분류**:
  - 🟢 **Optimal** (30-45 dB): Library Quiet - 도서관처럼 조용함
  - 🟡 **Good** (45-55 dB): Quiet Conversation - 조용한 대화 가능
  - 🟠 **Normal** (55-65 dB): Lively Chatter - 활기찬 대화 수준
  - 🔴 **Loud** (65+ dB): High Traffic - 시끄러운 환경
- **시각화**: 각 장소별로 평균 dB 값이 표시된 마커
- **필터링**: 소음 수준별로 필터링하여 원하는 장소만 표시 가능
- **즐겨찾기 강조**: 즐겨찾기한 장소를 별도 마커 스타일로 표시하는 개선 작업이 진행 중

#### 1.2 장소 검색
- **키워드 검색**: 카카오 로컬 API를 활용한 장소 검색
- **검색 결과 표시**: 검색 결과를 리스트로 표시하고, 선택 시 지도로 이동
- **자동완성**: 최근 검색어 기반 자동완성 기능
- **검색 기록**: 검색 기록 저장 및 관리 기능

#### 1.3 현재 위치 기반 서비스
- **현재 위치 표시**: 파란색 점으로 현재 위치 표시
- **위치 기반 검색**: 현재 위치를 중심으로 주변 장소 검색
- **지도 이동**: 현재 위치로 지도 이동 버튼

#### 1.4 날씨 정보
- **현재 날씨 표시**: 지도 왼쪽 상단에 날씨 위젯 표시
- **날씨 상세 정보**: 클릭 시 시간대별 날씨 예보 및 대기질 정보 표시

---

### 2. 소음 측정 및 리뷰 작성 (NewReviewActivity)

#### 2.1 실시간 소음 측정
- **측정 방법**: Android AudioRecord API를 사용한 실시간 소음 측정
- **측정 주기**: 50ms마다 업데이트되는 실시간 dB 값 표시
- **측정 결과**: 최종 측정된 dB 값과 소음 수준 분류 자동 표시
- **재측정 기능**: "Record Again" 버튼으로 언제든지 재측정 가능
- **개발 모드**: 디버그 빌드에서 데시벨 직접 입력 기능 제공

#### 2.2 리뷰 작성
- **별점 평가**: 1-5점 별점으로 장소 평가
- **텍스트 리뷰**: 자유롭게 리뷰 작성
- **이미지 첨부**: 갤러리에서 선택하거나 카메라로 직접 촬영하여 이미지 첨부
  - 여러 장의 이미지 업로드 가능
  - 이미지 미리보기 및 삭제 기능
  - Supabase Storage에 이미지 저장
- **리뷰 저장**: Supabase 데이터베이스에 리뷰 저장

#### 2.3 리뷰 수정
- **기존 리뷰 수정**: 작성한 리뷰의 별점과 텍스트 수정 가능
- **소음 수준 고정**: 측정된 소음 수준은 수정 불가 (데이터 신뢰성 유지)

---

### 3. 장소 상세 화면 (PlaceDetailActivity)

#### 3.1 장소 정보
- **장소명 및 주소**: 카카오맵 API에서 가져온 정확한 장소 정보
- **평균 소음 수준**: 해당 장소의 모든 리뷰를 기반으로 계산된 평균 dB 값
- **소음 수준 배지**: 색상별 배지로 소음 수준 표시
- **평균 평점**: 사용자들이 남긴 평균 별점 표시
- **리뷰 개수**: 총 리뷰 개수 표시

#### 3.2 Noise Trend Today 그래프
- **시간대별 소음 추이**: 최근 리뷰 데이터를 기반으로 한 소음 수준 그래프
- **시각화**: 라인 차트로 시간대별 소음 변화 표시
- **데이터 기반**: 실제 사용자들이 측정한 데이터로 구성

#### 3.3 리뷰 목록
- **리뷰 표시**: 해당 장소의 모든 리뷰를 최신순으로 표시
- **리뷰 정보**: 각 리뷰의 별점, 소음 수준, 작성 날짜, 리뷰 텍스트, 이미지 표시
- **이미지 확대**: 리뷰 이미지 클릭 시 확대 보기 기능

#### 3.4 즐겨찾기
- **즐겨찾기 추가/제거**: 하트 아이콘으로 즐겨찾기 관리
- **즐겨찾기 표시**: 즐겨찾기한 장소는 채워진 하트로 표시

---

### 4. 내 리뷰 관리 (MyReviewsActivity)

#### 4.1 리뷰 목록
- **내 리뷰만 표시**: 현재 로그인한 사용자가 작성한 리뷰만 표시
- **리뷰 정보**: 장소명, 소음 수준, 상태 배지, 평점, 작성 날짜, 리뷰 텍스트 표시
- **빈 리뷰 처리**: 리뷰가 없을 때 안내 메시지 및 새 리뷰 작성 버튼 표시

#### 4.2 필터링
- **소음 수준별 필터**: All Reviews, Library Quiet, Quiet Conversation, Lively Chatter, High Traffic
- **필터 적용**: 선택한 소음 수준의 리뷰만 표시

#### 4.3 정렬
- **정렬 옵션**:
  - Most Recent: 최신순
  - Highest Rating: 평점 높은 순
  - Optimal to Loud: 조용한 순서대로

#### 4.4 리뷰 수정 및 삭제
- **리뷰 수정**: 리뷰 항목 클릭 시 수정 화면으로 이동
- **리뷰 삭제**: 삭제 버튼 클릭 시 확인 다이얼로그 후 삭제
- **자동 갱신**: 수정/삭제 후 목록 자동 갱신

---

### 5. 즐겨찾기 관리 (MyFavoritesActivity)

#### 5.1 즐겨찾기 목록
- **즐겨찾기한 장소 표시**: 사용자가 즐겨찾기한 모든 장소 목록
- **장소 정보**: 장소명, 주소, 평균 소음 수준 표시
- **빈 목록 처리**: 즐겨찾기가 없을 때 안내 메시지 표시

#### 5.2 즐겨찾기 관리
- **즐겨찾기 해제**: 각 항목에서 즐겨찾기 해제 가능
- **장소 상세 이동**: 항목 클릭 시 장소 상세 화면으로 이동
